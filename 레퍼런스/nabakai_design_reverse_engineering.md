## 1. 디자인 시스템

### 1-1. 컬러 팔레트

```
배경 계층 (다크 기본):
  bg-base     #0a0a0c   ── 최외곽 배경 (manifest background_color)
  bg-surface  #18181b   ── 카드·패널 배경 (manifest theme_color)
  bg-elevated #27272a   ── 호버·포커스 상태

텍스트:
  text-primary   #f4f4f5  ── 주요 텍스트 (화이트에 가까운 회색)
  text-secondary #a1a1aa  ── 보조 설명, 캐시 시간 등
  text-muted     #52525b  ── 비활성, 레이블

상태 색상 (금융 도메인 표준):
  positive  #22c55e  ── 상승, 매수, 강세 (초록)
  negative  #ef4444  ── 하락, 매도, 약세 (빨강)
  neutral   #a1a1aa  ── 보합, 중립 (회색)
  warning   #f97316  ── 경고, 주의 (주황)
  caution   #eab308  ── 반대매매 위험, 노랑

강조 (브랜드):
  brand-primary  #6366f1  ── 인디고 (버튼, 링크, 점수 게이지 강조)
  brand-accent   #8b5cf6  ── 바이올렛 (AI 관련 요소)
```

### 1-2. 타이포그래피

```
폰트 패밀리:
  헤딩   Pretendard Variable (한국어 최적화 산세리프)
  본문   Pretendard Variable
  숫자   tabular-nums (숫자 정렬을 위한 모노스페이스 폭)
  코드   모노스페이스

크기 스케일:
  xs   12px / 0.75rem   ── 캐시 시간, 면책 조항
  sm   14px / 0.875rem  ── 보조 레이블, 테이블 셀
  base 16px / 1rem      ── 기본 본문
  lg   18px / 1.125rem  ── 섹션 부제목
  xl   20px / 1.25rem   ── 섹션 제목
  2xl  24px / 1.5rem    ── 페이지 제목
  3xl  30px / 1.875rem  ── 지수 숫자 (코스피값 등)

숫자 표기 규칙:
  지수값      소수점 2자리, 천 단위 콤마
  변화율      +/- 부호 강제, 소수점 2자리, % 접미사
  거래대금    억원 단위 (1조 = 10,000억), 소수점 1자리
  USD 금액    $B 단위 (수십억) 또는 $M (수백만)
```

### 1-3. 간격 & 그리드

```
기본 단위: 4px (0.25rem)

컨테이너: max-width 1280px, 좌우 패딩 16px (모바일) / 24px (태블릿) / 32px (데스크톱)

카드 패딩: 12px (sm) / 16px (base) / 20px (lg)
섹션 간격: 24px (모바일) / 32px (데스크톱)

그리드:
  모바일  1열
  태블릿  2열
  데스크톱 3~4열 (대시보드 카드)
```

### 1-4. 공통 컴포넌트 규칙

```
카드:
  - 배경: bg-surface (#18181b)
  - 테두리: border-zinc-800 1px
  - 라운드: rounded-xl (12px)
  - 그림자: shadow-lg (다크 배경에서 미세하게)

테이블:
  - 헤더: bg-zinc-900, text-zinc-400
  - 홀수행: bg-surface
  - 짝수행: bg-zinc-900 (약간 어두운 줄무늬)
  - hover: bg-zinc-700

태그/칩:
  - 배경: bg-zinc-800
  - 텍스트: text-zinc-300
  - 라운드: rounded-full
  - 패딩: px-2 py-0.5

점수 게이지:
  - 0~29: 빨강 (risk)
  - 30~49: 주황 (caution)
  - 50~69: 노랑 (neutral)
  - 70~84: 초록 (positive)
  - 85~100: 밝은 초록 (strong)
```

## 2. 전역 레이아웃 & 네비게이션

### 2-1. 헤더 구조

```
┌──────────────────────────────────────────────────────────┐
│  [나박AI 로고]  대시보드  오늘시황  큰손보유  전체콘텐츠  소개  [🌙][KR]  │
└──────────────────────────────────────────────────────────┘

- 위치: sticky top-0, z-index 높음
- 배경: bg-zinc-950/90 + backdrop-blur (반투명)
- 높이: 56px (모바일) / 64px (데스크톱)
- 로고: "나박AI" 텍스트 + 브랜드 컬러, 클릭 시 / 이동
- 메뉴: 데스크톱에서 텍스트 링크, 모바일에서 햄버거
- 우측: 다크모드 토글(🌙/☀️) + 언어 전환(KR/EN)
- 세션 배지: 헤더 우측에 "장 중 🟢" 또는 "장마감 ⚫" 실시간 표시
```

### 2-2. 메인 탭바 (대시보드 전용)

```
┌──────────────────────────────────────────┐
│  [한눈에]  [테마]  [AI 랩]  [더보기]      │
└──────────────────────────────────────────┘

- 위치: 헤더 아래 sticky
- 배경: bg-zinc-900
- 활성 탭: 하단 border-b-2 brand-primary + text-white
- 비활성 탭: text-zinc-400, hover시 text-zinc-200
- 탭 전환: 스크롤 위치 최상단으로 reset
- 모바일: 4개 탭 균등 분할 (각 25%)
```

## 3. 메인 대시보드 — 한눈에 탭

### 3-1. 히어로 섹션 (최상단)

```
┌──────────────────────────────────────────────────────┐
│  코스피  8,108.59  ▲ +6.02%          [세션: 주말]    │
│  코스닥    869.33  ▲ +0.30%                          │
│  USD/KRW 1,384.5  ▼ -0.15%                          │
│                                                      │
│  내일 갭 예상: [보합 ━]   바이낸스 한국주: [실시간]  │
└──────────────────────────────────────────────────────┘

설계 원칙:
- 3개 지수를 한 줄씩 대형 폰트(3xl)로 표시
- 변화율: 양수=초록, 음수=빨강, 보합=회색
- 방향 아이콘: ▲▼ 또는 화살표 아이콘
- 세션 배지: 우측 상단 고정 (애니메이션 pulse)
- 업데이트: 5초마다 숫자가 부드럽게 업데이트 (transition)
```

### 3-2. 공포·탐욕 지수 & 장세 나침반

```
┌─────────────────────┬──────────────────────────┐
│   공포·탐욕 지수     │      장세 나침반          │
│                     │                          │
│     [게이지]         │   추세 56 ●──────        │
│      46점            │   시장폭 30 ●──          │
│     "보통"           │   수급  62 ●────         │
│                     │   심리  46 ●───          │
│  [6개 지표 탭]        │   신용  38 ●──           │
│  • 모멘텀   58       │   거시  52 ●───          │
│  • 신고가   32       │                          │
│  • VIX     61       │  종합: 46점 "신호 혼조"  │
└─────────────────────┴──────────────────────────┘

설계:
- 좌: 반원 게이지 SVG (0~100, 색상 그라데이션)
- 우: 방사형 차트 또는 바 형태 6축 표시
- 탭 클릭으로 구성 지표 상세 토글
- 색상: 점수 구간별 자동 변경
```

## 4. 메인 대시보드 — 테마 탭

### 4-1. 오늘 강한 테마 (가로 막대 차트)

```
┌──────────────────────────────────────────────────┐
│  오늘 자금이 향한 곳          [외인+기관 억원]   │
│                                                  │
│  반도체   ████████████████ +2,840억              │
│  2차전지  ████████████     +1,920억              │
│  바이오   █████████        +1,450억              │
│  방산     ██████            +830억               │
│  게임     ████              +610억               │
│                                                  │
│  ← 막대 클릭 시 해당 테마 상세 펼쳐짐            │
└──────────────────────────────────────────────────┘

설계:
- 수평 막대 그래프 (최대값 기준 100% 스케일)
- 색상: 양수=초록, 음수=빨강
- 클릭 인터랙션: 아코디언으로 세부 종목 표시
- 막대 위에 호버 시 정확한 금액 툴팁
```

```typescript
function ChangeText({ pct }: { pct: number }) {
  const color = pct > 0 ? 'text-green-400' : pct < 0 ? 'text-red-400' : 'text-zinc-400'
  const sign  = pct > 0 ? '+' : ''
  const arrow = pct > 0 ? '▲' : pct < 0 ? '▼' : '━'
  return (
    <span className={`${color} tabular-nums`}>
      {arrow} {sign}{pct.toFixed(2)}%
    </span>
  )
}
```
