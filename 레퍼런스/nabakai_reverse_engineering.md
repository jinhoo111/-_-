## A. 핵심 데이터 엔드포인트 구조

### `/flow.json` — 메인 데이터 허브

nabakai.com의 핵심 실시간 데이터는 단일 엔드포인트 `/flow.json`에서 모두 공급된다.  
서비스워커(`sw.js`)가 이 URL을 캐시 제외 목록에 등록하여 **항상 최신 데이터를 직접 요청**한다.

```
GET https://nabakai.com/flow.json
Cache: never (서비스워커가 /api/* 및 /flow.json 캐시 차단)
응답: ~200KB JSON
갱신: 30초 자동 갱신 (클라이언트 polling)
```

**최상위 키 구조 (21개):**

| 키 | 타입 | 데이터 | 상태 |
|----|------|--------|------|
| `ts` | string | ISO 타임스탬프 | live |
| `session` | string | "OPEN" / "CLOSED" / "PRE" / "POST" | live |
| `session_phase` | object | 세션 단계 상세 | live |
| `demo` | bool | 데모 모드 여부 | — |
| `hero` | object | 코스피/코스닥 요약 지표 | live |
| `news` | array | AI 생성 뉴스 | live |
| `ai_commentary` | string | AI 시황 해설 텍스트 | live |
| `insider_buys` | array | DART 임원·대주주 매수 | live |
| `lockup_expiry` | array | 락업 만료 예정 종목 | live |
| `short_squeeze` | array | 공매도 스퀴즈 신호 종목 | live |
| `supply_anomaly` | array | 수급 이상 감지 종목 | live |
| `technicals` | object | 기술적 신호 스캐너 | live |
| `global_market` | object | 글로벌 지수·환율·암호화폐 | live |
| `monday_forecast` | object | 월요일 예측 (AI) | demo |
| `us_market` | object | 미국 지수·섹터·52주 신호 | live |
| `us_kis_rank` | object | KIS 미국주식 거래 순위 | ok (fdr_fallback) |
| `us_owner` | object | Form4 임원거래 + 구루 포트폴리오 | live |
| `theme_flow` | object | 테마 자금 흐름 | stale |
| `_meta` | object | 각 섹션별 freshness 상태 | — |
| `_status` | string | 전체 응답 상태 | — |

**`session_phase` 객체 구조:**

## B. KIS (한국투자증권) API 연동

### 연동 방식

KIS(한국투자증권) Open API를 백엔드 서버에서 직접 폴링하여 데이터를 수집한다.  
`_source: "kis_realtime_5s"` 필드로 5초 주기 실시간 수집임을 명시한다.

### 연동 데이터 목록

#### 1. 국내 지수 실시간 (5초 주기)

**`hero.kospi`, `hero.kosdaq`, `hero.kospi200`**

```json
{
  "code": "KS11",
  "last": 8108.59,
  "change": 459.85,
  "change_pct": 6.02,
  "open": 7702.75,
  "high": 8154.88,
  "low": 7700.12,
  "prev_close": 7648.74,
  "volume": 1234567890,
  "value": 98765432100,
  "_updated_at": "2026-07-04T11:36:00",
  "_source": "kis_realtime_5s"
}
```

| 필드 | 설명 |
|------|------|
| `code` | KIS 내부 종목/지수 코드 (`KS11`=코스피, `KQ11`=코스닥, `KS200`=코스피200) |
| `last` | 현재가 |
| `change` / `change_pct` | 전일 대비 변화량/변화율 |
| `open`, `high`, `low` | 당일 시가/고가/저가 |
| `prev_close` | 전일 종가 |
| `volume` / `value` | 거래량 / 거래대금 |

#### 2. 수급 이상 감지 (`supply_anomaly`)

KIS 장중 투자자별 순매수 데이터를 20일 평균과 비교하여 이상 수급 포착.

```json
{
  "code": "000150",
  "name": "한미약품",
  "main": "기관",
  "today_eok": 224,
  "avg20_eok": 45,
  "ratio": 4.97,
  "kind": "buy",
  "price": 345000,
  "pct": 2.3,
  "price_live": true
}
```

| 필드 | 설명 |
|------|------|
| `main` | "외국인" 또는 "기관" |
| `today_eok` | 당일 순매수 (억원) |
| `avg20_eok` | 20일 평균 순매수 (억원) |
| `ratio` | `today_eok / avg20_eok` — 이상 배수 |
| `kind` | "buy" / "sell" |
| `price_live` | 실시간 가격 여부 |

#### 3. 미국 주식 거래 순위 (`us_kis_rank`)

KIS 해외주식 API에서 미국주식 거래대금/거래량 순위 수집.  
KIS 장애 시 `FDR(FinanceDataReader)` 폴백 적용 (`"_source": "fdr_fallback"`).

**배열 구조 (value_rank, volume_rank, gainers, losers, price_change 공통):**

```json
{
  "ticker": "MU",
  "name": "마이크론",
  "ename": "Micron Technology Inc",
  "excd": "NAS",
  "market": "NAS",
  "last": 118.52,
  "rate": -9.84,
  "tvol": 87654321,
  "tamt": 603300000000,
  "tamt_b": 10.39,
  "pbid": 118.50,
  "pask": 118.55,
  "rank": 1,
  "value_usd": 10390000000,
  "volume": 87654321,
  "change_pct": -9.84
}
```

| 필드 | 설명 |
|------|------|
| `excd` | 거래소 코드 (`NAS`=나스닥, `NYS`=뉴욕증권거래소) |
| `tamt` | 거래대금 (원화 환산) |
| `tamt_b` | 거래대금 (십억 달러) |
| `pbid` / `pask` | 매수호가 / 매도호가 |

#### 4. 프로그램매매 추이

KIS 데이터로 코스피·코스닥 프로그램 순매수 장중 시간대별 누적 추이를 추적.  
차익거래 / 비차익거래 분류하여 표시.

#### 5. 매물대 (Volume Profile)

KIS 일봉 데이터 60거래일치를 받아 가격대별 거래량 분포 계산.  
현재가 기준 상단 저항선까지 거리(%), 하단 지지선까지 거리(%) 산출.

## C. DART (금융감독원 공시) API 연동

### 연동 방식

DART Open API (`https://opendart.fss.or.kr/`)를 통해 임원·대주주 주식 거래 공시를 수집.  
공시 직링크는 `https://dart.fss.or.kr/dsaf001/main.do?rcpNo={접수번호}` 형식.

### 연동 데이터: `insider_buys` 배열

대주주·임원의 자사주 매수/매도 공시 데이터.

```json
{
  "code": "011780",
  "name": "금호석유화학",
  "exec": "국민연금공단",
  "report_nm": "주식등의대량보유상황보고서",
  "shares": 1234567,
  "price": 89000,
  "amount_eok": 3205,
  "hold_rate": 12.34,
  "hold_rate_delta": 0.85,
  "time": "2026-07-03",
  "trade_first": "2026-07-01",
  "trade_last": "2026-07-03",
  "trade_days": 3,
  "rcept_no": "20260703001234",
  "dart_url": "https://dart.fss.or.kr/dsaf001/main.do?rcpNo=20260703001234",
  "cap_ratio_pct": 12.34,
  "shares_ratio_pct": 0.85
}
```

| 필드 | 설명 | DART 원본 |
|------|------|-----------|
| `rcept_no` | 공시 접수번호 | DART 고유키 |
| `dart_url` | 공시 원문 링크 | `dsaf001/main.do` |
| `report_nm` | 공시 보고서명 | "주식등의대량보유상황보고서" 등 |
| `hold_rate` | 보유 지분율(%) | 공시 원본값 |
| `hold_rate_delta` | 지분율 변화 | 계산값 |
| `trade_first` / `trade_last` | 거래 시작/종료일 | 공시 원본 |
| `cap_ratio_pct` | 시가총액 대비 거래금액 비율 | 계산값 |

**UI 연동:**
- `hero.insider_count`: 당일 공시 건수 (30건)
- `hero.insider_eok`: 당일 공시 금액 합계 (1,022억원)
- `dart_url`을 클릭하면 DART 공시 원문으로 직접 이동

## D. KRX (한국거래소) 데이터 연동

### 연동 방식

KRX 통계 서비스 (data.krx.co.kr) 또는 장마감 후 확정 데이터를 수집.  
`신용잔고`는 마감 확정 후 18시에 갱신 (`"신용융자 잔고: 18시 갱신"`으로 UI에 명시).

### 연동 데이터 목록

#### 1. 공매도 비중 추세 (`short_squeeze`)

14거래일 공매도 비중 변화 추적 + 스퀴��� 신호 판별.

```json
{
  "code": "096770",
  "name": "SK이노베이션",
  "short_change_pct": 8.4,
  "price_pct": -2.1,
  "short_ratio": 12.5,
  "squeeze": false,
  "note": "공매도 증가 중",
  "hist_change_pct": 15.2,
  "hist_trend": "up",
  "hist_days": 5
}
```

| 필드 | 설명 |
|------|------|
| `short_change_pct` | 5일간 공매도 비중 변화율 |
| `price_pct` | 주가 변화율 |
| `short_ratio` | 현재 공매도 비중(%) |
| `squeeze` | 스퀴즈 조건 충족 여부 |
| `hist_trend` | 역사적 추세 방향 |
| `hist_days` | 추세 지속 일수 |

**스퀴즈 조건 (유추):** `short_ratio` 높은데 `price_pct` 상승 = 숏커버링 압력 발생

#### 2. 신용융자 잔고

- 신용잔고 총액 (조 단위)
- 반대매매 발생액 (억 단위)
- 60영업일 추세 그래프용 시계열 데이터
- 종목별 신용잔고비율 (빚투 TOP 순위)

#### 3. 대차잔고 급변

- 대차잔고 급증 종목 (5일 내 급증 = 공매도 선행 신호)
- 대차잔고 급감 종목 (숏커버 가능성)
- 데이터 소스: `KIS` 명시 (KRX 원본을 KIS가 재제공하는 경로로 추정)

#### 4. 투자자별 순매수 (외국인·기관·개인)

- 국장 개인 순매수 TOP (30분 캐시)
- 외국인·기관 순매수 상위/하위
- KRX 가집계 (장중) vs 확정치 (장마감 후) 구분 표시

## E. SEC (미국증권거래위원회) 데이터 연동

### 연동 방식

SEC EDGAR 공개 API (`https://www.sec.gov/cgi-bin/browse-edgar`) 또는  
13F / Form4 HTML 파싱을 통해 기관·임원 거래 내역 수집.

### 연동 데이터 1: Form4 임원 거래 (`us_owner.form4`)

```json
{
  "ticker": "AMZN",
  "company": "Amazon.com Inc",
  "owner": "Andy Jassy",
  "title": "CEO",
  "action": "SELL",
  "total_usd": 1498845,
  "shares": 5667,
  "price": 263.95,
  "remain": 2209666,
  "date": "2026-06-28"
}
```

| 필드 | SEC 원본 대응 |
|------|--------------|
| `owner` | Form4 신고자명 |
| `title` | 직책 (CEO, CFO, CTO 등) |
| `action` | "BUY" / "SELL" |
| `total_usd` | 총 거래금액 |
| `remain` | 신고 후 보유 주식 수 |
| `date` | Form4 제출일 |

### 연동 데이터 2: 13F 분기 공시 (`us_owner.gurus`)

워런 버핏, 캐시 우드 등 대형 기관투자자 분기 포트폴리오 공시.  
분기말 기준 45일 내 SEC 제출 → 나박AI 수집 → 고래(Whale) 페이지에 표시.

```json
{
  "guru_id": "buffett",
  "name": "워런 버핏 (버크셔 해서웨이)",
  "total_aum": 263100000000,
  "filed": "2026-05-15",
  "top10": [
    {
      "symbol": "애플",
      "ticker": "AAPL",
      "weight": 22.0,
      "value_usd": 57800000000,
      "shares": 300000000
    }
  ],
  "top_all": [...]
}
```

| 필드 | 설명 |
|------|------|
| `guru_id` | 내부 식별자 (buffett, ark, dalio 등) |
| `total_aum` | 운용 자산 총액 (USD) |
| `filed` | 13F 제출일 |
| `weight` | 포트폴리오 비중(%) |
| `value_usd` | 보유 평가액 (USD) |

**고래(Whale) 페이지 연동:**  
`/prerender/whale/{guru_id}.html` — 250개+ 기관/인물별 개별 프리렌더 페이지  
대상: 버핏, ARK, Dalio, Burry, Ackman, 르네상스, Druckenmiller, Soros 등 해외 20+ 기관 + 한국 기관 200+ 곳

## F. FDR (FinanceDataReader) 폴백 연동

### 연동 방식

KIS API 장애 또는 데이터 미수신 시 Python 라이브러리 `FinanceDataReader(FDR)` 로 폴백.  
`us_kis_rank._source: "fdr_fallback"` 및 `fetched_at` 타임스탬프로 폴백 발동 여부 확인 가능.

```json
"_source": "fdr_fallback",
"fetched_at": "2026-07-04T01:30:19.278801+00:00"
```

**FDR 제공 데이터:**
- 미국 주식 거래대금 / 거래량 / 가격 변동 순위
- 상승/하락 상위 종목

## G. 암호화폐 데이터 연동 (Binance 등)

### 연동 방식

`global_market.crypto` 배열에서 암호화폐 실시간 가격 표시.  
`_source: "primary"` 표기이며 `_ticker` 필드 없음 → Binance 또는 CoinGecko API 직접 호출 추정.  
바이낸스 한국주(Binance Korean Stocks) 선물 데이터도 별도 수집.

### 연동 데이터: `global_market.crypto`

```json
[
  { "sym": "BTC", "label": "비트코인", "price": 63450.2, "pct": 1.32, "_source": "primary" },
  { "sym": "ETH", "label": "이더리움", "price": 3210.5, "pct": 0.85, "_source": "primary" },
```

## I. AI 생성 콘텐츠 파이프라인

### 연동 방식

수집된 데이터를 기반으로 LLM(Claude 또는 GPT 계열 추정)이 자동으로 텍스트 생성.  
결과물을 `flow.json` 내 텍스트 필드로 내보내거나 별도 프리렌더 HTML로 저장.

### 생성 콘텐츠 목록

#### 1. `ai_commentary` (시황 해설)

```
"기관의 SK하이닉스·삼성전자 대규모 매수와 외국인의 차익 실현 매도가 
교차하는 가운데, 국민연금의 금호석유화학 블록딜 3,205억원이 
장중 수급에 영향을 미쳤습니다..."
```

- 당일 `insider_buys`, `supply_anomaly`, `hero` 데이터를 프롬프트로 주입
- 서술형 문장으로 변환하여 반환

#### 2. `news` 배열 (뉴스 브리핑)

```json
{
  "source": "NabakAI",
  "url": "https://t.me/coinaiai_report/12345",
  "msg_id": 12345
}
```

- 텔레그램 채널(@coinaiai_report)과 연동
- `msg_id`로 텔레그램 메시지 직접 링크
- 외신 뉴스를 수집해 AI가 한국어 브리핑으로 요약 생성

#### 3. `monday_forecast` (주간 예측)

```json
{
  "direction": "up",
  "confidence": "중",
  "summary": "미국 장 +1.2%, 반도체 호조...",
  "key_drivers": [
    { "label": "엔비디아 +4.21%", "impact": "반도체 호조", "dir": "up" }
  ]
}
```

- 주말에 다음 주 월요일 코스피 방향 예측
- `key_drivers` 배열로 근거 항목화
- `confidence`: "상" / "중" / "하"
- 현재 `_meta`에서 "demo" 상태

#### 4. `/prerender/us_report.html` (미국 AI 리포트)

매일 1종목 AI 선정 + 분석 보고서 자동 생성. 프리렌더 HTML로 저장.

생성에 사용하는 데이터:
- 종가, SPY 대비 상대강도, 저항 수준, 손절선
- 거래량 증가율, RSI, P/E
- VIX, CPI YoY, 장단기 스프레드
- 선정 점수 (0~100)

#### 5. `/prerender/stock/{code}.html` (종목 AI 요약)

KIS + DART + KRX 데이터를 종합하여 종목별 AI 분석 텍스트 생성 후 프리렌더 저장.

#### 6. Battle 예측 (`/battle`)

매일 코스피 시가 예측 (상승/하락/보합 방향 + 변동률 %).  
이전 거래일 결과와 비교하여 적중 여부 자동 판별 및 기록.

## J. 내부 데이터 파이프라인 & 아키텍처

### 아키텍처 추정 (공개 정보 기반)

```
외부 API (KIS / DART / KRX / SEC / FDR / Binance 등)
           ↓ (백엔드 봇 / 스케줄러)
     데이터 수집 & 계산 서버
           ↓
     /flow.json (정적 파일 or 동적 엔드포인트)
           ↓
   클라이언트 (SPA) 30초마다 fetch polling
           ↓
     렌더링 (탭별 컴포넌트)
```

**프리렌더 파이프라인:**
```
백엔드 봇 (일 1회 or 정기 실행)
  → 데이터 수집 및 AI 생성
  → /prerender/*.html 정적 파일 생성
  → CDN/정적 서버 배포
```

### 내부 계산 로직 (추정)

#### 나박지수 (0~100)

5개 항목 합산 후 당일 리스트 내 순위 기반 정규화:
1. 외국인 순매수 (KIS)
2. 기관 순매수 (KIS)
3. 오너·임원 매수 (DART)
4. 공매도 비중 역수 (KRX)
5. 테마 자금 유입 (KIS)

각 항목 `-2 ~ +2` 점수 → 합산 → 0~100 정규화

#### 장세 나침반 (0~100)

6개 지표 가중 합산:
- 추세 축 (이동평균 배열): KIS 일봉 데이터
- 시장폭 축 (상승/하락 종목 비율): KRX
- 심리 축 (공포탐욕지수): CNN/자체 계산
- 신용·과열 축 (RSI + 신용잔고): KIS + KRX
- 수급 축 (외인·기관 방향): KIS
- 거시 축 (미국 지표): FRED + 선물

#### 기술적 신호 스캐너 (`technicals`)

| 신호 | 계산 방법 | 데이터 소스 |
|------|-----------|------------|
| `golden_day` | 5일선 > 20일선 교차 완료 | KIS 일봉 |
| `golden_soon` | \|5일선 - 20일선\| < 임계값 | KIS 일봉 |
| `golden_week` | 주봉 5선 > 주봉 20선 | KIS 주봉 |
| `high_52w` | 현재가 ≥ 52주 최고가 × 0.99 | KIS |
| `cci_oversold` | CCI ≤ -100 | KIS OHLCV |
| `above_w20` | gap_pct = (현재가 - 20주선) / 20주선 | KIS 주봉 |

#### 공매도 스퀴즈 감지 (`short_squeeze`)

```
squeeze = True 조건 (추정):
  short_ratio > 임계값 (예: 5%)
  AND price_pct > 0 (주가 상승 중)
  AND short_change_pct < 0 (공매도 비중 감소 = 숏커버)
```

#### 수급 급반전 감지

```
조건: 
  어제_순매수_방향 != 오늘_순매수_방향
  AND 거래대금 > 평균 이상
```

#### 프렉탈 (과거 유사 패턴)

수급·추세·테마 지표를 벡터화 → 과거 3년 데이터에서 코사인 유사도로 유사 구간 검색 →  
유사 구간 이후 15거래일(3주) 수익률 분포 산출 → 순위화하여 표시.
